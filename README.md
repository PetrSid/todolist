# 📝 ToDoList

Небольшое веб-приложение для управления повседневными задачами. Позволяет быстро добавлять дела, отмечать их выполненными и удалять ненужные.

## ✨ Функционал

- ➕ Добавление новых задач
- ✅ Отметка о выполнении (зачеркивание)
- 🗑️ Удаление задач
- 💾 Сохранение данных в LocalStorage (задачи не пропадут после обновления страницы)

## 🛠 Технологии

Проект создан с использованием:

*   **HTML5** — структура страницы
*   **CSS3** — стили и адаптивность
*   **JavaScript (ES6+)** — логика приложения

## 🚀 Как запустить

1.  Скачайте или склонируйте репозиторий:
    ```bash
    git clone https://github.com/[ТВОЙ_НИК]/[НАЗВАНИЕ_РЕПОЗИТОРИЯ].git
    ```

2.  Откройте файл `index.html` в любом браузере.
    *(Или используйте Live Server в VS Code)*
